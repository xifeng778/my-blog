# Promise

一种异步编程的解决办法

axios和fetch都有使用promise来实现

三种状态

pending（进行中）

resolve（已成功）

reject（已失败）

# 基本用法

const promise = new Promise(function(resolve, reject) {
  // ... some code
  if (/* 异步操作成功 */){
    resolve(value);
  } else {
    reject(error);
  }
});
